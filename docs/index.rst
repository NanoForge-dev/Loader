NanoForge Loader
================

``@nanoforge-dev/loader`` provides runtime loaders for the NanoForge game engine.
It enables serving, loading, and executing NanoForge games in both browser and
server environments with hot-reload support for development.

The project consists of three packages:

- **loader-client**: Browser-based loader with web interface
- **loader-server**: Server-side loader with worker thread support
- **loader-website**: Interactive web interface for game loading
